PROJETOS

Gerador de Senhas

Este projeto foi realizado para que os usuários possam criar senhas seguras para o seu uso pessoal. O usuário poderá escolher a quantidade e os tipos de caracteres para gerar a sua senha forte.

Relógio digital

O projeto consiste na elaboração de um relógio digital que possui um fundo animado e além disso, ele informa a data completa para o usuário.

Link tree

Voltado para realizar uma divulgação prática de todas as suas redes sociais, o link tree, é um projeto elegante que consiste em um banner,com um fundo dinâmico, que possue o link de redes sociais do usuário, recomendado para divulgação profissional.

Cronômetro

O projeto consiste na criação de um cronômetro com as funções para inciar, parar, continuar e resetar.
